
# MCP Server - Guide d'Utilisation

## 📋 Vue d'ensemble

Ce document décrit l'intégration et l'utilisation du serveur MCP (Model Context Protocol) dans le projet immobilier IA.

## 🚀 Démarrage rapide

### Prérequis
- python 3.10+

### Installation
```bash
    pip install fastmcp
```
##  Utilisation
```bash
    python main_mcp.py
```

##  Structure du module mcp_server

```
mcp_server/
├── tools/
├── services/
├── main_mcp.py
└── README_mcp.md
```
